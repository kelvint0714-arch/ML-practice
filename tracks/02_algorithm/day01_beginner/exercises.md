这份练习面向刚开始学习 Python 的同学。目标不是让你立刻独立写出机器学习程序，而是让你能够：

- 看懂 Day 1 的主要数据流；
- 认出代码中的变量、函数、参数和返回值；
- 说清楚 `X`、`y`、`fit`、`predict` 和评价指标；
- 判断一个模型是否可能过拟合；
- 知道哪些内容今天必须理解，哪些工程代码可以以后再学。

## 使用规则

1. 先做题，再查看文末答案。
2. 小实验请放在一个新建的临时 Notebook 中，不要改原始 Notebook。
3. 所有实验只在内存中创建临时变量，不写 CSV、JSON 或其他文件。
4. 不要修改或运行带有 `to_csv()`、`write_text()` 的保存代码。
5. 不要使用测试集训练、调参或反复查看结果。
6. 如果不确定某一步会不会修改仓库，先停下来问 Codex。

本练习把内容分成三类：
- **必须理解**：完成 Day 1 前必须能用自己的话解释；

# 常见错误排查

| 现象 | 常见原因 | 安全处理 |
|---|---|---|
| `NameError` | 前面的变量定义没有运行 | 从临时 Notebook 第一格按顺序重新运行 |
| `SyntaxError` | 括号、引号、冒号不配对 | 对照示例逐个检查符号 |
| `IndentationError` | 函数或循环缩进不一致 | 统一使用四个空格，不混用 Tab |
| `ModuleNotFoundError: numpy` | 选错 Kernel 或环境缺包 | 先检查当前 Kernel，不要随意重装全部环境 |
| 缺少 PyTorch/TensorFlow 的提示 | DeepChem 在检查可选模块 | Day 1 若后续继续运行，通常不影响本实验 |
| `AssertionError` | 实际值与预期条件不一致 | 不要删除检查；记录报错行并告诉 Codex |
| shape 看不懂 | 混淆样本数和特征数 | 记住二维 `X` 通常是 `(行数, 特征数)` |
| R² 是负数 | 模型可能比简单基线更差 | 这不一定是代码错误，结合 Dummy 和数据划分解释 |
| 结果与截图小数略有不同 | 环境、版本或计时不同 | 先比较数据规模、随机种子和主要指标口径 |
| 不小心改了原 Notebook | 在原文件中做了练习 | 先不要保存或提交，告诉 Codex检查差异 |
| 结果文件时间或内容变化 | 运行了保存单元 | 不要自行回滚或删除，告诉 Codex处理 |
| 单元没有输出 | 只有赋值或函数定义 | 这是正常的；用 `print()` 或在最后一行写变量名观察 |
